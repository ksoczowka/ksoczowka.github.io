Desk Slime

I'm currently working on a project called Desk Slime, where I leverage the raylib library and incorporate features from modern C++.

The project centers around a base class Pet, which allows for easy customization.

Additionally, I've designed the Pet class in such a way that it can be utilized as a library by including the pet.hpp header and linking with the corresponding pet.cpp file in other projects.

This modular design promotes code reuse and flexibility across different applications.
